Welcome

Anki Web Browser is installed. Check out below the most recent updates.

Updates

  • Edit notes (cards) directly during the review
    • It's possible to add (only) texts in the review stage
  • Web Browser attached to main Window (Dock bar)
    • Instead of using a standalone window, it can be in multiple positions in the Anki Window
    • To change it, go the Web Browser configurations
  • Use Session between executions (cookies are enabled and persisted between executions)
    • This allows the user to login in sites and the session will remain in the next time using Anki
  • Technical changes:
    • Modified integration with Anki
    • The original integration is legacy. This new way aims to reduce un-expected bugs
    • Change integration with DarkReader
    • Tough still unstable

Be patient

The current version had several changes in the baseline code. It might require some time for stabilization.
In case of issues, please report it at: https://github.com/ssricardo/anki-web-browser/issues
